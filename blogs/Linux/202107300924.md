## install ffmpeg
```Shell
sudo apt-get update
sudo apt install ffmpeg
```

## convert wmv to mp4
```shell
sudo ffmpeg -i v1.wmv -qscale:v 1 v1.mp4
sudo ffmpeg -i v2.wmv -qscale:v 1 v2.mp4
```


## concat vedio
```shell
sudo vim mylist.txt
file 'v1.mp4'
file 'v2.mp4'

ffmpeg -f concat -i mylist.txt -c copy output.mp4
```
