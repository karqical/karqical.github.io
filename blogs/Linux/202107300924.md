## install ffmpeg
```Shell
sudo apt-get update
sudo apt install ffmpeg
```
## concat vedio
```shell
sudo vim mylist.txt
file 'v1.mp4'
file 'v2.mp4'

ffmpeg -f concat -i mylist.txt -c copy output.mp4
```
